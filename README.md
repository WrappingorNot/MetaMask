
# MetaMask

## 오퍼스엠 안드로이드 과제

## 구현 기능 

 1. 메인 화면 구현

 2. 스왑 화면 구현

 3. 각 메뉴 클릭 기능, 클릭이벤트시 Dialog 플로팅 구현 (Custom Dialog, BottomSheetDialog, Fragment, RecyclerView)

 4. 메인화면에서 스왑화면 전환 기능 구현(Intent)

 5. Retrofit2를 사용한 업비트 API 연결 구현 ( market, ticker api사용 )
